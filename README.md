# SIDHKOFED Website Revamp

Agile, reviewable implementation of the SIDHKOFED public portal revamp.

This first increment is a static, bilingual-ready prototype so stakeholders can
review information architecture, homepage flow, navigation, and visual direction
before the CMS/backend stack is locked.

## Open Locally

Open `index.html` in a browser.

## Current Increment

- Sprint 0 documentation and audit notes
- Sprint 1 visual foundation
- Sprint 2 homepage MVP
- Sprint 3 representative core public pages in a single-page shell

## Key Paths

- `index.html` - public portal prototype
- `src/css/main.css` - design system and responsive layout
- `src/js/app.js` - language toggle, navigation, accessibility controls, search/filter UI
- `assets/images/` - prototype visual assets
- `docs/agile-backlog.md` - sprint backlog and acceptance criteria
- `docs/codex-infra-handover.md` - implementation handover context
- `docs/project-structure.md` - prototype structure and conventions
- `docs/sidhkofed-cms-codex-context.md` - full CMS scope mirrored from the shared OneDrive context
- `docs/cms-integration-conventions.md` - compact CMS naming, model, field, and API conventions
- `docs/api-context/` - module-based API context for future CMS backend implementation
- `docs/progress-log.md` - multi-device progress tracker and handoff log
